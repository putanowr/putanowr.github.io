Lumped d/dt brick ( ). Add the standard discretization of a first order time derivative. This version of the brick uses lumped mass matrix. The parameter $rho$ is the density which could be omitted (the defaul value is 1). This brick should be used in addition to a time dispatcher for the other terms.
