Detailed Description

@brief CircHoleQuarterParams – parameters describing 1/4 of a rectangular domain with circular hole.

Handle the following parameters: Lx – rectangle dimension in X direction Ly – rectangle dimension in Y direction Rh – hole radius Nx – number of boundary segments in X direction Ny – number of boundary segments in Y direction Nh – number of boundary segments at the hole
