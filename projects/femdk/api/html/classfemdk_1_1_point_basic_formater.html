Detailed Description

This is a helper class to allow writing generic routines for saving point data to a stream. Instead of putting the formating code directly into the loops that iterate over points and their coordinates, the formating rules are encapsulated in PointBasicFormater class and the functions dealing with points are parameterized by object of this class.

It is assumed that a point data section have the following structure:

{header} {prefix}{ index_1 }{c_1}{sep}{c_2}{sep}...{sep}{c_n}{suffix} {prefix}{ index_2 }{c_1}{sep}{c_2}{sep}...{sep}{c_n}{suffix} ............. {footer}

where c_1,c_2, – are point coordinates sep – coordinate separator

PointBasicFormater allows user to set the strings for: header, prefix, separator, suffix and footer. Additionally it stores two boolean flags: point_indices – if true point indices are written do_not_validate – if true the routine write_point do not check validation flag on the point.

Additionally it provides methods for formating point records, writing point indices, writing footer and header. These methods are declared as virtual and can be easily overwriten in classes derived form PointBasicFormater.
