Detailed Description

@brief GnuplotParams – parameters for making graphs with gnuplot.

Handle the following parameters: gptCommands – vector of commands gpt_keep_datafiels – boolean, if true do not delete auxiliary data files gpt_mesh_file – name of file for saving mesh data
