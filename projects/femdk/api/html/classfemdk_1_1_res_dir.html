◆ cd()

void femdk::ResDir::cd ( void  ) const

Calling this method changes the current working directory of the calling process to the directory with path storied in this object.

Can be safely called multiple times.

In order to return to the original directory one can call cd_back().

If some of the underlying system calls (like mkdir or chdir) fails, throws iso::bas_failure exception.
