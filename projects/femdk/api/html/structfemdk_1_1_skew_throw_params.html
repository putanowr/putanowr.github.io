Detailed Description

@brief SkewThrowParams – parameters for skew-throw problem

Handle the following parameters: st_alpha – throw angle (degrees) st_v0 – initial velocity st_x0 – initial position X-coordinate st_y0 – initial position Y-coordinate st_dt – time step
