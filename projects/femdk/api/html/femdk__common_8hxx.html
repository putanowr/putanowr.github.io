◆ FEMDK_STANDARD_MAIN

#define FEMDK_STANDARD_MAIN (   ParamsType,
  optionsName,
  customLogger 
)

Standard FEMDK main procedure. It uses AutoOpts generated code to parse command line arguments into an instance of ParamsType. The name of AutoOpts structure for parsed optins is set to {optionsName}Options. Besides parsing command line options this routine initializes and finalizes FEMK framework by calling femdk::Initialize and femdk::Finalize, respectively. User code should be placed in the function int main_body(int argc, char **argv, ParamsType &params);
