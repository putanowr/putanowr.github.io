Detailed Description

@brief CubeMeshParams – mesh parameters for cube mesh

Handle the following parameters: Nx – mesh density (number of segments) in X direction Ny – mesh density (number of segments) in Y direction Nz – mesh density (number of segments) in Z direction

If setup from parsed command line, the above parameters can be specified by –Nx, –Ny and –Nz flags or by –mesh-resolution flag which takes a vector. The –mesh-resolution values can be overwritten by specifying resolution in any specific direction.
