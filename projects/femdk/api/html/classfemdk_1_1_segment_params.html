Detailed Description

@brief SegmentParams – parameters describing geometric segments

Handle the following parameters: segment_begin – string with begin point coordinates, segment_end – string with end point coordinates segment_at – explicit point at segment segment_lambda – segment paramter

The coorindate strings are not interpreted nor validated by this class.
