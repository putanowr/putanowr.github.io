Detailed Description

This class defines a handler for SAX parser of FEBio data.

This class provides methods that are called at particular events while parsing FEBio XML data. The main responsibility of this class is to parse XML data into proxy containers. An instance of this class does not utilise the parsed data but instead it delegates handling of the parsed data to a converter object by which it is parameterised.

Users wishing to convert FEBio data to some specific format should pass a suitable FEBioConverter object to the constructor.
